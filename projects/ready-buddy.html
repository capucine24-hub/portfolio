Project Overview

Ready Buddy is an educational app designed for children aged 6 to 14. It helps them learn how to react in everyday emergencies through short, interactive games.

This version focuses on the Junior Squad (6–10). The app mixes game design, storytelling and simple UI so that survival skills feel fun, clear and memorable.

This prototype was built using SwiftUI, with one game developed in Unity and integrated into the iOS app.

From Thesis to Prototype

Ready Buddy was developed as my Master's thesis project, exploring how gamification can improve survival education for children.

I handled the full process, from research on children's learning and UI/UX for kids to designing the visual identity and building the complete iOS app.

This project taught me how to structure larger codebases, debug complex integrations, and balance educational goals with playful design.

Ready Buddy Website

I also created a simple Ready Buddy website to support parents and testers. It explains the project, shows the visual identity, and includes an early chatbot prototype that uses a squad-specific database (currently for the Junior Squad) to help children learn safety skills through friendly guided questions.
